This program can be run through the main method of the Restaurant class. The program takes 2 parameters, the first being an integer value 1-3 which corresponds to the seating strategy, the second is an integer value 1-2 which corresponds to the waiting list strategy. It is important to note that the program does not check/ have exception handling built in for invalid input types. Should an invalid input be entered, or no inputs at all for a given argument then exceptions will be thrown by java.
